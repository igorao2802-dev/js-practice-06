📘 Подсказки

  • querySelector / querySelectorAll — основной способ поиска элементов. querySelector возвращает первый найденный, querySelectorAll — NodeList всех.
  • textContent vs innerHTML — для пользовательских данных всегда используйте textContent. innerHTML допустим только для статичного, заранее известного HTML.
  • createElement + append — создавайте узлы через JS, не через строки HTML.
  • classList.toggle / add / remove — управляйте внешним видом через классы, а не через прямое изменение style.
  • element.remove() — удаляет элемент из DOM напрямую, без ссылки на родителя.
