📋 Требования к выполнению

  • Все обработчики событий — через addEventListener в script.js
  • Пользовательский текст вставляется только через textContent (защита от XSS)
  • Валидация: название не может быть короче 3 символов
  • При удалении всех карточек — подтверждение через confirm()
  • Комментарии к каждому createElement, append, remove, classList
  • Использовать querySelector / querySelectorAll для поиска элементов
